# Design

Referências visuais e diretrizes de design do RespFrias.

## Arquivos

- `respfrias-dashboard-reference.png` — imagem de referência do dashboard (salvar manualmente)

## Diretrizes

- Mobile-first obrigatório
- Cores primárias: `#0d7ea8` (azul-teal)
- Botões grandes para uso no celular
- Cards com sombra leve, bordas arredondadas
- Linguagem simples, sem jargão técnico
- Status visuais claros (badges coloridos)
